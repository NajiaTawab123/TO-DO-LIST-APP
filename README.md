# TO-DO List App 📝

A simple and responsive To-Do List App built with HTML, CSS, and JavaScript.

### 🌟 Features:
- Add & delete tasks
- Toggle light/dark theme
- Persistent storage using Local Storage
- Responsive UI for all devices

### 🚀 Technologies Used:
- HTML
- CSS
- JavaScript
